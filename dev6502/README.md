# dev6502

Docker image for 6502 development, mainly for commodore machines.

* High level languages 
    * C (GCC)
    * C++ (G++)
    * Java (OpenJDK11)
    * Python 3

* Assembly compilers
    * ASL
    * KickAss 5.23
    * CC65
    * ACME
    * 64TASS
    * DASM

* Loaders
    * Krill's Loader v184

* Utils
    * Exomizer 2.0.11, 3.0.2, 3.1.1
    * PuCrunch
    * cc1541

* Other untils
    * make
    * nano, vim, mc
    * grep, sed, awk
    * git, subversion, wget
    * zip, unzip, gzip
